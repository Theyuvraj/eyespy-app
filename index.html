EyeSpy — How to Use
1
Select Class & Due Date
Pick the class. Due date defaults to tomorrow — change if needed.
2
Add Subjects
Tap "＋ Add Subject", select a subject, type the homework. Add as many subjects as needed.
3
Save & Print
Tap Save. EyeSpy saves to Google Sheets and opens the label preview. If content is too long, a warning shows — split into a second sticker.
📐 Min font is 6.5pt. EyeSpy warns if content won't fit.
4
School Logo on Label
Your THPS logo and school name appear on every printed label automatically.
1
Select Class & Date → Load
Pick the class and date being checked, then tap Load Checklist.
2
Mark Not-Done Students
All students start as Done (green ✓). Tap the toggle to mark a student Not Done (red). Subject chips appear — tap to specify which subjects are missing.
🎯 The WhatsApp message groups by subject automatically based on your chip selection.
3
Save → WhatsApp Message
Tap Save. The WhatsApp message is generated with completed subjects listed first, then not-done students grouped by subject.
4
Late Submission
Tap "View Pending Students" to see previously not-done students. When they submit late, tap Resolved to update their record.
1
Heat Map
Month at a glance — green = high completion, red = low. Hover for exact %.
2
Student Report
Search any student, tap to see full year record, subject breakdown, and completion heat map.
3
Parent Report Card
Select class, month, student. Add optional teacher note. Print a professional parent-facing card with logo, completion rate, and subject breakdown.
1
Bulk Import Students
Settings → Bulk Import → select class → paste names or upload CSV. App adds all at once.
2
WhatsApp Template
Settings → WhatsApp Template → edit format. Use variables: {DATE}, {CLASS}, {NOT_DONE_LIST}, {COMPLETED_LIST}, {SCHOOL_NAME}.
3
Label Size
Default is 60 × 40 mm. Change to match your printer paper. Min font stays at 6.5pt.
1
Create Google Sheet
Go to sheets.google.com → create blank sheet → copy the Sheet ID from the URL.
2
Deploy Backend Script
script.google.com → New Project → paste code (Settings → Get Script Code) → replace YOUR_SHEET_ID → Deploy as Web App → Execute as Me → Anyone.
3
Connect EyeSpy
Settings → paste URL → Test Connection → done. All data now syncs to your Sheet.
💡 Cost: ₹0/month. Runs on your free Google account.
